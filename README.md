# MERN BOOKSTORE APP
## Run Backend Server
1. cd backend 
2. npm install
3. npm run dev
## Run Frontend
1. cd FE
2. npm install (`npm install socket.io-client` if error relavants to socket, otherwise ignore this)
3. Change API link with your IP address in FE\constants\api.js
4. npx expo
## Run Socket Server
1. cd socket 
2. npm install
3. npm run dev
## Run Admin
1. cd admin
2. npm install
3. npm run dev
